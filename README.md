<h2 align="center">Get some issues from Github</h2>

This app use GraphQL api of github.com for showing some issues and fun.

<h2 align="center">Installation</h2>


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
